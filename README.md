## SQL Data Analytics & Exploratory Analysis Framework

This project focuses on performing structured exploratory data analysis (EDA) using SQL on relational datasets. It demonstrates how raw data can be explored, transformed, and analyzed to extract meaningful insights.

### Key Features:
- Exploration of dimensions and measures for structured analysis
- Aggregation and ranking queries (Top-N, Bottom-N)
- Time-based trend analysis and change-over-time insights
- Data segmentation and performance analysis
- Query-based reporting for decision-making

### Tech Stack:
- SQL (Core Analysis)
- Relational Databases
